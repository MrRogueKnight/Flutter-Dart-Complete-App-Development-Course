### 2️⃣ In a Flutter project, which file is primarily responsible for launching the app?

✔️ **Answer**: `main.dart`

### 3️⃣ What feature allows immediate reflection of code changes in the running Flutter app without fully restarting it?

✔️ **Answer**: **Hot Reload**
